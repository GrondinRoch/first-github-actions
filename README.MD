# Projet GitHub Actions CI/CD

Ce projet est dédié à la mise en place des pipelines CI / CD avec GitHub Actions.

Il contient des fichiers de configuration permettant d'automatiser les tests, la validation et le déploiement du code sur GitHub.

Objectifs :
- Configurer l’intégration continue (CI)
- Automatiser le déploiement continu (CD)
- Utiliser GitHub Actions pour exécuter des workflows à chaque push ou pull request

Ce repository sert d'exemple pour apprendre à créer et gérer des actions GitHub dans un projet.
